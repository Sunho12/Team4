# 📊 고객 행동 기반 예측 점수(Scoring) 모델 명세서

## 1. 모델 개요

본 모델은 고객의 **과거 패턴(기변 주기)** 및 **현재 상태(데이터 사용량)**와 **능동적 의사(상담 이력)**를 결합하여 이탈 및 변경 가능성을 수치화합니다. **시그모이드(Sigmoid) 함수**를 기본 골격으로 하되, 특정 임계점 도달 전 상담 발생 시 **로그(Log) 기반 가중치**를 부여하여 잠재 수요를 조기에 포착합니다.

---

## 2. 공통 수학 모델

모든 점수는 **0~100점** 사이로 산출되며, 기준점() 도달 여부에 따라 가중치를 차등 적용합니다.

### ① 기본 점수: 시그모이드 함수 ()

* : 분석 변수 (경과일 또는 사용량)
* : 평행이동 기준점 (50점이 산출되는 지점)
* : 곡선의 기울기 (민감도)

### ② 상담 가중치: 밑이 2인 로그 ()

상담 내 특정 키워드 언급 횟수()에 대해 가중치를 산출합니다. 상용로그()보다 초기 반응에 민감하게 반응합니다.


* **특징:** 단 1회의 언급만으로도 가중치가 **2.0**이 되어 점수가 두 배로 상승합니다.

---

## 3. 모델별 상세 파라미터

### [모델 1] 기기 변경 예측 (Time-based)

고객의 교체 주기를 분석하여 기변 가능 시점을 예측합니다.

* **변수 ():** 마지막 기기 변경 후 현재까지의 경과일
* **기준점 ():**
* 기변 이력 2회 이상: 해당 고객의 **평균 기변 주기** (개인화)
* 기변 이력 1회 이하: **730일** (표준 2년)


* **기울기 ():** **0.07**
* 기준일() 도과 후 약 30일 이내에 점수가 90점에 도달하는 강력한 타겟팅 설정



### [모델 2] 요금제 변경 예측 (Usage-based)

데이터 사용량 한계에 따른 요금제 상향(Up-selling) 가능성을 예측합니다.

* **변수 ():** 당월 데이터 사용량 (GB)
* **기준점 ():** 요금제별 속도 제한(QoS) 임계치
* **300kbps**: 3GB
* **1Mbps**: 7GB
* **3Mbps**: 15GB


* **기울기 ():** **1.1**
* 임계치() 대비 약 2GB를 추가 사용할 때 점수가 90점에 도달하는 설정



---

## 4. 로직 핵심 가이드 (Quick Summary)

| 구분 | 기변 예측 모델 | 요금제 변경 모델 |
| --- | --- | --- |
| **핵심 변수** | 시간 (Day) | 사용량 (GB) |
| **기준점()** | 개인별 주기 (또는 730) | 요금제별 QoS (3 / 7 / 15) |
| **기울기()** | **0.07** (매우 가파름) | **1.1** (적정 민감도) |
| **가중치()** | 기변 관련 키워드 상담 수 | 요금제 관련 키워드 상담 수 |
| **전략** | 교체 주기 임박 자 집중 관리 | 속도 제한 불편 고객 식별 |

---

## 5. 기대 효과

1. **초정밀 타겟팅:** 일반적인 일괄 기준이 아닌 고객의 과거 패턴()에 기반한 개인화 점수 산출
2. **조기 반응 포착:** 아직 주기가 도래하지 않았더라도 상담 신호()가 있다면 즉각 마케팅 대상으로 분류
3. **데이터 정규화:** 로그 함수와 Max-cap() 설정을 통해 점수 왜곡 방지
